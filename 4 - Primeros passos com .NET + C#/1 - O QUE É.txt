### CONHECENDO O FRAMEWORK

#### Passado, Presente e Futuro do .NET?

Criado pela microsoft no ano 2000, com o nome de NEXT GENERATION WINDOWS SERVICES (NGSWS).                                                                             

ASP.X extensão dos arquivos web forms do .NET



#### O que é, como e onde usar o .NET?

É uma infraestrutura para desenvolvimento de software criado pela microsoft.

E uma aplicação .NET é desenvolvida para e roda em uma das seguintes implementações do .NET

	- .NET CORE (Focar aqui .NET CORE com C#)
	- NET FRAMEWROK
	- MONO (XAMARIN)
	- UNIVERSAL WINDOWS PLATAFORM (UWP)



#### Como?

Cada implementação inclui um ou mais .NET Runtimes (ambientes de execução)

- .NET CORE: CoreCLR e CoreRT

- .NET FRAMEWORK: CLR

- MONO: Mono Runtime

- UWP: .NET Native

  

  

#### **Atualmente a Microsoft desenvolve e suporta 3 linguagens para .NET**

C#, F# e VB
